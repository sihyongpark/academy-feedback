# Changelog

## 2026-03-15

### 추가
- 로그인 세션 유지 — 새로고침 후에도 자동 로그인 유지 (localStorage), 로그아웃 시 삭제
- 학생 생년월일(birth_date) 필드 — 테이블, 모달 폼, 상세 편집, Excel 양식/가져오기 포함
- 학생 원생구분 컬럼 — 재원/퇴원 뱃지 표시
- 학생 목록 필터 — 재원생 / 퇴원생 / 전체 (기본: 재원생)
- 문자 발송 드래프트 자동 저장 — localStorage에 저장, 새로고침 후에도 유지

### 수정
- 문자 발송 드래프트 복원 시점 수정 — 마운트 시 records 미로딩으로 드래프트 복원 안 되던 버그 수정
- Excel 양식 다운로드 버튼 레이블 → "Excel 양식 다운로드"
- 학생 목록 정렬 기준 → enrolled_at (등록일) 오름차순
- 메시지 생성 프롬프트에서 하드코딩 규칙 제거 — 관리자 가이드라인에 전적으로 따르도록 변경
- 문자 발송 진행 중 편집/재작성 비활성화, 발송 완료 후 드래프트 삭제
